# TTY Command Practice

## What is the `tty` Command?

The `tty` command in Unix/Linux is used to tell you **which terminal (or pseudo-terminal)** you are using.

---

## Common `tty` Command Usage:

- **`tty`**  
  Prints the terminal device file you're currently using.

---

## Problems and Answers:

### (i) Display the Current Terminal Name
**Question:**
Find out which terminal you are using.

**Answer:**
```bash
tty
```
### Explanation:

- The output will be something like:
```bash
/dev/ttys000
```
